BoundingLayoutAnchor

public struct BoundingLayoutAnchor

The BoundingLayoutAnchor type is a factory decorator for creating grouped NSLayoutConstraint objects using a fluent interface (an interface designed to produce more readable code). Use these constraints to programatically define your layout using Auto Layout.
