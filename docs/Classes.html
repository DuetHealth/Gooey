  • SafeAreaLayoutGuide

    The SafeAreaLayoutGuide is a custom layout guide designed to provide backwards-compatible references to a view controller’s safe area. On iOS 10 and below, this uses the top and bottom layout guides. On iOS 11 and above, this uses the view’s safe-area layout guide.
